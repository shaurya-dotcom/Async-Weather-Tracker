# 🌤 Async Weather Tracker

## Description
This is a simple weather application built using HTML, CSS, and Vanilla JavaScript.  
It fetches real-time weather data from the OpenWeatherMap API using asynchronous programming.

## Features
- Search weather by city name
- Displays temperature, condition, humidity, and wind speed
- Uses async/await and fetch()
- Error handling with try...catch
- Stores search history using Local Storage
- Demonstrates event loop using console logs

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeatherMap API

## How to Run
1. Generate your API key from OpenWeatherMap.
2. Replace the API key inside `script.js`.
3. Open `index.html` in your browser.

## Concepts Covered
- Async / Await
- Promises
- Fetch API
- Event Loop
- Local Storage
